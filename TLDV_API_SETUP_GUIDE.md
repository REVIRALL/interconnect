# 🔑 TL;DV API キー取得ガイド（初心者向け）

## 📌 TL;DVって何？
TL;DVは、ZoomやGoogle Meet、Microsoft Teamsの会議を自動で録画して、文字起こし（トランスクリプト）してくれるサービスです。

## 🚀 APIキー取得の手順

### 1️⃣ TL;DVにログイン

1. **ブラウザを開く**（Chrome、Safari、Edgeなど）

2. **TL;DVのサイトにアクセス**
   ```
   https://tldv.io
   ```

3. **ログインする**
   - 画面右上の「Log in」ボタンをクリック
   - メールアドレスとパスワードを入力
   - 「Sign in」をクリック

   ![ログイン画面のイメージ]
   ```
   ┌─────────────────────────┐
   │  📧 Email               │
   │  [あなたのメール]        │
   │                         │
   │  🔒 Password            │
   │  [••••••••]             │
   │                         │
   │  [ Sign in ]            │
   └─────────────────────────┘
   ```

### 2️⃣ 設定画面を開く

1. **プロフィールメニューを開く**
   - ログイン後、右上の丸いアイコン（プロフィール画像）をクリック
   
2. **設定を選択**
   - メニューから「Settings」をクリック

   ```
   ┌─────────────────┐
   │ 👤 Your Profile │
   │ ⚙️ Settings     │ ← これをクリック
   │ 🚪 Log out      │
   └─────────────────┘
   ```

### 3️⃣ API設定に移動

1. **左側のメニューを確認**
   - 「API」または「Developer」という項目を探す
   - クリックして開く

   ```
   Settings
   ├── 👤 Profile
   ├── 🔔 Notifications
   ├── 💳 Billing
   ├── 🔑 API         ← これをクリック
   └── 🛡️ Security
   ```

### 4️⃣ APIキーを生成

1. **「Generate New API Key」ボタンを探す**
   - 青い大きなボタンがあるはず
   - クリックする

2. **キーの名前を入力**
   ```
   ┌─────────────────────────────────┐
   │ API Key Name:                   │
   │ [INTERCONNECT連携用]            │ ← わかりやすい名前を入力
   │                                 │
   │ [ Generate ] [ Cancel ]         │
   └─────────────────────────────────┘
   ```

3. **「Generate」をクリック**

### 5️⃣ APIキーをコピー

⚠️ **超重要！** APIキーは一度しか表示されません！

1. **表示されたAPIキーをコピー**
   ```
   ┌─────────────────────────────────────────┐
   │ Your API Key:                           │
   │                                         │
   │ tldv_sk_1234567890abcdefghijk...      │
   │                                         │
   │ [ Copy to Clipboard ]                   │
   └─────────────────────────────────────────┘
   ```

2. **必ずメモ帳などに保存**
   - コピーボタンをクリック
   - メモ帳を開いて貼り付け（Ctrl+V または Cmd+V）
   - ファイル名「TL;DV_APIキー.txt」などで保存

### 6️⃣ プロジェクトに設定

1. **設定ファイルを作成**
   - `env-example.txt`をコピー
   - `.env`という名前に変更

2. **APIキーを貼り付け**
   ```
   変更前：
   TLDV_API_KEY=ここにTL;DVのAPIキーを貼り付け
   
   変更後：
   TLDV_API_KEY=tldv_sk_1234567890abcdefghijk...
   ```

## ❓ よくある質問

### Q: APIキーが見つからない
A: TL;DVの無料プランではAPI機能が制限されている場合があります。有料プランへのアップグレードが必要かもしれません。

### Q: APIキーを忘れてしまった
A: セキュリティのため、一度生成したAPIキーは再表示できません。新しいキーを生成してください。

### Q: 料金はかかる？
A: APIの使用には通常、有料プランの契約が必要です。料金プランは以下を確認：
- Free: API利用不可
- Pro: 月額$20〜（API利用可能）
- Business: カスタム料金

### Q: セキュリティは大丈夫？
A: APIキーは絶対に他人に教えないでください！GitHubなどに公開しないよう注意。

## 🛠️ トラブルシューティング

### エラー：「Invalid API Key」
→ APIキーが正しくコピーされているか確認
→ 前後に余分なスペースがないか確認

### エラー：「API Access Denied」
→ 有料プランに加入しているか確認
→ API機能が有効になっているか確認
