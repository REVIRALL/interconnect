## 📍 ステップ6: 入力設定（Start）

### やること
1. **「Start」ノード**をダブルクリック
2. 右側に設定パネルが表示される

### 入力項目を追加
```
┌─────────────────────────────────────┐
│ 入力変数                            │
│ ┌─────────────────────────────────┐ │
│ │ + 変数を追加                    │ │ ← クリック
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

### 3つの変数を追加

#### 1つ目：transcript
- **変数名**: `transcript`
- **タイプ**: `テキスト`（Text）
- **必須**: ✅ チェックを入れる

#### 2つ目：user_id
- **変数名**: `user_id`
- **タイプ**: `テキスト`（Text）
- **必須**: ✅ チェックを入れる

#### 3つ目：analysis_type
- **変数名**: `analysis_type`
- **タイプ**: `テキスト`（Text）
- **デフォルト値**: `interest_extraction`

## 📍 ステップ8: LLMノードを設定

### LLMノードをダブルクリック

#### 1. モデル選択
```
モデル：[▼ GPT-4 を選択]
```

#### 2. プロンプト入力
下の大きなテキストボックスに以下をコピペ：

```
あなたは会議トランスクリプトを分析する専門家です。
以下のトランスクリプトから、ビジネスマッチングに役立つ情報を抽出してください。

トランスクリプト：
{{transcript}}

以下のJSON形式で出力してください：
{
  "keywords": [重要なキーワードを10-20個],
  "topics": [主要なトピックを5-10個],
  "interests": [
    {
      "category": "カテゴリ名（technology/marketing/finance等）",
      "keywords": ["関連キーワード"],
      "strength": 0.1-1.0の数値
    }
  ],
  "business_areas": ["関連するビジネス分野"],
  "summary": "200文字程度の要約"
}

注意事項：
- 日本語のキーワードも英語のキーワードも抽出する
- strengthは話題に出た頻度や重要度で判断
- ビジネスに関連する内容を重視
```

## 📍 ステップ9: ノードを接続

### やること
1. **Start**ノードの下の●をクリック
2. そのまま**LLM**ノードの上の●までドラッグ
3. **LLM**ノードの下の●をクリック
4. そのまま**End**ノードの上の●までドラッグ

### 完成図
```
┌─────────────┐
│   Start     │

│     │       │
│     ↓       │
│     ●       │

│     ●       │
│     │       │
│     ↓       │
│     ●       │
│    End      │
```
