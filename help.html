プロフィール情報を変更するには？

プロフィール情報の変更は以下の手順で行えます：

  1. 左側のメニューから「プロフィール」をクリック
  2. 「編集」ボタンをクリック
  3. 変更したい項目を編集
  4. 「保存」ボタンをクリック

イベントに参加するには？

イベントへの参加方法：

  1. 「イベント」ページを開く
  2. 参加したいイベントを選択
  3. 「参加する」ボタンをクリック
  4. 必要事項を入力して申込完了

パスワードを忘れた場合は？

パスワードをリセットする手順：

  1. ログイン画面の「パスワードを忘れた方」をクリック
  2. 登録済みのメールアドレスを入力
  3. 送信されたメールのリンクをクリック
  4. 新しいパスワードを設定

退会するには？

退会をご希望の場合は、以下の手順でお手続きください：

  1. 「設定」ページを開く
  2. 「アカウント」タブを選択
  3. ページ下部の「退会する」をクリック
  4. 退会理由を入力して確認

※退会するとデータは復元できません
